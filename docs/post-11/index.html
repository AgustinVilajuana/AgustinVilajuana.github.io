10 - Modelado 3D de Superficies

Acá se realiza paso a paso la creación de un solido por medio de una serie de superficies en el programa Fusion 360.

Pasos a seguir

  • Lo primero que haremos es crear un boceto con la forma de base que tendra nuestro solido mas adelante.

  • Ahora haremos un plano de desfase, el cual pondremos a la altura deseada sobre el otro boceto.
  • Sobre este dibujaremos otro boceto.

  • Ahora haremos un loft con ambas superficies.

  • Realizaremos un nuevo boceto pero en vista frontal.
  • Luego usaremos el comando extruir para darle profundidad a la nueva superficie.

  • Ahora con el nuevo cuerpo haremos un split body para crear un agujero en la otra superficie.

  • Con el comando recortar haremos que la segunda superficie quede solamente lo que se encuentra al interior de la primera.

  • Con patch cerraremos las caras.
  • Con stich vamos a unir las superficies para obtener un solido.

  • Con empalme redondearemos los bordes de todo nuestro solido.

  • Con Shell generaremos un vaciado.
  • De esta forma toma la caracteristica de caja y con un grosor de 3mm.

  • Ahora inspeccionamos el elemento para ver bien su forma, haciendo un análisis de sección.
