Precious Plastic

Precious-Plastic es un Open Source donde el fin es el reciclaje del plástico, de manera más efectiva, donde todos pueden acceder a la información necesaria para poder lograrlo.

Es una iniciativa mundial, y por medio de la tecnología, se crea conciencia del daño que provoca el plástico al ecosistema.

De manera online y de forma gratuita, las personas pueden acceder a un set de 40 videos instructivos, libros guía y todo el conocimiento necesaria para poder tener un espacio de trabajo completamente funcional de reciclaje de plásticos.

Este Open Source permite a las personas no solo ver esta infromación, si no que tambien poder realizar pregutas y tambien aportar con conocimientos propios. También existen mapas, donde las personas pueden ver quienes estan trabajando en el mismo campo, y de esa forma crear grupos locales para así dar soluciones locales.

Make It Precious

La misión de este movimiento es darle un nuevo valor a los plásticos. Aca es donde la palabra “precious” toma importancia y entre todos, podremos eliminar el desperdicio de este material. De esta forma, esta iniciativa invita a las personas a crear emprendimientos y negocios en base al desperdicio de plásticos.

La verdad es que encuentro que este tema es muy interesante, ya que, puede llevar a las personas a utilizar sus propios residuos plásticos en objetos necesarios para sus casas, artículos para el desarrollo de actividades personales o incluso para la venta de estos. Esto creo que Re-valora el plástico, dandole un fin el cual no es solo contaminación.
