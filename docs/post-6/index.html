5 - Impresión 3D

Acá se nos pidio simular una impresion 3D con el software Cura

Cura

Este software funciona como un slicer, y nos permite transformar lo objetos modelados en distintos programas como Rhino o Fusion 360 a un archivo de código G, el cual permite a la impresora realizar lo necesario para imprimir el modelo. Es muy fácil de usar y además es compatible con muchas impresoras con sus distintas configuraciones. También no estima costos y tiempos de impresión por lo que es muy útil y versátil.

Simulación molde en Cura

  • Lo primero que debemos hacer es definir que es lo que queremos imprimir.
  • En mi caso es un molde el cual usaremos para dar forma al Caucho granulado con un adhesivo.

  • Luego exportamos este archivo como stl para abrirlo en Cura

  • Ahora elegiremos la impresora a usar y el material para la impresión.
  • Tambien definiremos el tamaño del cabezal.
  • Utilizaremos la impresora Endler 3, un cabezal de 0,4 mm y como material el PLA morado para notar bien el contraste y relieve del modelo.

  • Luego en la configuración de impresión le daremos una distancia de 0,3 mm entre cada capa.

  • Ahora en la parte superior pasamos de “Prepare” a “Preview”.
  • Acá podemos ver la simulación de la impresion con la barra que se ve a la derecha, capa por capa.
  • También podemos ver el tiempo que demorara esta impresón.

Y así es como podemos llevar a cabo una simulación de impresión para luego realmente imprimir lo que deseemos. En este caso en específico podemos ver que el molde no posee caracteristicas de un buen desmoldaje, es por esto que volvimos a modelar el objeto y realizamos nuevamente el proceso, el cual se encuentra actualmente siendo impreso en el fablab de la universidad.

Podemos ver que este molde es mucho mas grande y requiere de mucho mas tiempo y material. Así es cómo con Cura podemos ver y entender todo esto de manera muy simple.
