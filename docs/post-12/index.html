11 - Insertar iFrame

Acá se realizara paso a paso como insertar in iFrame en nuestros sitios personales.

Pasos a seguir

  • Primero que todo debemos guardar lo que sea que estemos modelando con fusion 360 en la carpeta llamada Admin Project.
  • Esta carpeta esta conectada a internet y queda guardada automaticamente en nuestro archivos de autodesk en la web.

  • Ahora nos vamos a la pagina web http://myhub.autodesk360.com.
  • Ingresamos a nuestro usuario de Autodesk para poder buscar el modelo que queremos insertar.

  • Nos vamos a la carpeta Admin Project que fue donde guardamos el modelo.
  • Le damos click al modelo que queremos instertar.

  • Hacemos click en el icono de compartir.
  • En incrutar copiamos el codigo iFrame que nos aparece para luego usarlo en visual studio code, para el post que nosotros deseemos.

  • Ahora utilizaremos el siguente codigo:

“{{< iframe-fusion link=” " >}}"

  • Dentro de las comillas pondremos nuestro codigo de iFrame.
  • Es muy importante que dentro de las comillas solo pegamos desde “https” hasta “embed”, el resto que nos copia desde autodesk no nos importa para insertar el iFrame.
